 <h1>Chat-App</h1>
    <p>A real-time chat application built with React, Node.js, and Tailwind CSS.</p>

    <h2>Key Features &amp; Benefits</h2>
    <ul>
        <li><strong>Real-time Messaging:</strong> Instant message delivery using Socket.IO.</li>
        <li><strong>User Authentication:</strong> Secure user accounts and login functionality.</li>
        <li><strong>Responsive Design:</strong> Tailwind CSS provides a responsive layout that adapts to different screen sizes.</li>
        <li><strong>Modern UI:</strong> Clean and intuitive user interface using React components.</li>
    </ul>

    <h2>Prerequisites &amp; Dependencies</h2>
    <p>Before you begin, ensure you have the following installed:</p>
    <ul>
        <li><strong>Node.js:</strong> Download and install the latest LTS version from <a href="https://nodejs.org/" target="_blank">nodejs.org</a>.</li>
        <li><strong>npm:</strong> Typically installed with Node.js.</li>
        <li><strong>Git:</strong> For cloning the repository.</li>
    </ul>

    <p>The project relies on these key dependencies:</p>
    <ul>
        <li>React</li>
        <li>React DOM</li>
        <li>React Router DOM</li>
        <li>Tailwind CSS</li>
        <li>Axios</li>
        <li>Js-cookie</li>
        <li>Socket.IO Client</li>
        <li>Vite</li>
        <li>Eslint</li>
    </ul>

    <h2>Installation &amp; Setup Instructions</h2>
    <ol>
        <li><strong>Clone the repository:</strong>
            <pre><code>git clone https://github.com/Mukesh1765/Chat-App.git
cd Chat-App</code></pre>
        </li>
        <li><strong>Navigate to the client directory:</strong>
            <pre><code>cd client</code></pre>
        </li>
        <li><strong>Install client-side dependencies:</strong>
            <pre><code>npm install</code></pre>
        </li>
        <li><strong>Start the development server:</strong>
            <pre><code>npm run dev</code></pre>
            <p>This will typically start the development server on <code>http://localhost:5173</code>. Check the console output for the exact URL.</p>
        </li>
        <li><strong>Server-side setup:</strong>
            <p>If there is server side code, make sure to install the server side dependencies and run the server. The server is commonly in the root directory of the project.</p>
        </li>
    </ol>
